
# Kalkulator

Sebuah Kalkulator sederhana menggunakan bahasa dart


## Run Locally

Clone the project

```bash
  git clone https://github.com/Pashakhatamihasibuan/kalkulator-dart.git
```

Pergi ke project directory

```bash
  cd kalkulator-dart
```

Pastikan Dart SDK sudah terinstal

```bash
  install Dart
```

Jalankan Dart

```bash
  dart main.dart
```
